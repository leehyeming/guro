# 서비스 정의서 — 구로 관광 홈페이지
> 작성일: 2026-06-02 | 작성자: 이혜민 | 단계: ITCONNECT Pipeline Step 1

---

## 1. 서비스 한 줄 요약

> **"숲에서 비우고 버들마을에서 채우다" — 구로구 항동·오류동 반나절 힐링 코스를 소개하는 관광 홈페이지**

---

## 2. 타깃 고객

| 항목 | 내용 |
|------|------|
| 주요 타깃 | 일상 속 틈새 힐링이 필요한 **20~30대 커플, 1인, 가족** |
| 방문 목적 | 도심 속 자연 힐링 + 로컬 맛집 탐방 |
| 추천 시간대 | 평일 오후, 주말 오전~오후 |
| 예상 소비 | 1인 약 2만 원 |

---

## 3. 고객의 문제

- 구로구에 **6만 평 규모 수목원**(푸른수목원)이 있지만 **존재 자체를 모르는 사람이 많다**
- 항동철길, 오류버들마을 등 **로컬 명소들이 흩어져 있어** 묶어서 즐기는 동선을 찾기 어렵다
- 네이버/구글 검색으로는 **개별 장소 정보만 나오고**, 코스형 여행 정보는 없다

---

## 4. 대체재 (현재 방문객이 쓰는 방법)

- 네이버 블로그 (정보 산발, 최신성 불분명)
- 인스타그램 해시태그 탐색 (사진만 있고 실용 정보 부족)
- 구로구청 공식 관광 페이지 (딱딱하고 UI가 불편)

---

## 5. 우리 서비스의 해결책

| 문제 | 해결 |
|------|------|
| 코스 동선을 모른다 | 지도 + 번호 순서로 한눈에 시각화 |
| 각 장소가 어떤 곳인지 모른다 | 장소별 카드 (사진 + 매력 포인트 + 팁) |
| 밥은 어디서 먹냐 | 오류버들마을 맛집 섹션 따로 구성 |
| 공유하고 싶은데 뭐라 쓰지? | SNS 해시태그 복사 + 공유 버튼 |

---

## 6. 핵심 기능 (MVP)

### 반드시 있어야 하는 것 ✅
1. **히어로 섹션** — 코스명 + 핵심 카피 + "코스 보러가기" 버튼
2. **코스 개요 카드** — 소요시간, 예상비용, 이동수단, 추천 타깃
3. **동선 지도** — 5개 장소 시각적 지도 (번호 + 이름 + 이동시간)
4. **장소별 소개** — 6개 장소 카드 (사진 자리 + 매력포인트 + 방문 팁)
5. **맛집 & 먹거리** — 오류버들마을 대표 메뉴 (감자옹심이, 수제버거, 통닭 등)
6. **방문 정보** — 교통 안내, 주의사항 (그늘 없음, 킥보드 금지 등)
7. **SNS 공유** — 해시태그 모음 + 카카오/인스타 공유 버튼
8. **헤더 네비게이션** — 섹션 앵커 링크 (코스 소개 / 장소 / 맛집 / 가는 법)

### 나중에 추가할 것 📌
- 실제 사진 교체 (현재는 플레이스홀더)
- 카카오맵 API 연동 (현재는 정적 지도 이미지)
- 다른 팀원 코스 추가

---

## 7. 페이지 구조 (홈페이지 섹션 순서)

```
[헤더 네비게이션]
    로고 | 코스 소개 | 장소 | 맛집 | 가는 법

[① 히어로]
    배경 이미지 + 코스명 + 한 줄 카피 + CTA 버튼

[② 코스 개요]
    아이콘 카드 4개: 소요시간 / 예상비용 / 이동수단 / 추천 타깃

[③ 추천 동선 지도]
    정적 지도 or SVG 일러스트
    장소 1~5 번호 + 이름 + 이동시간 표시

[④ 장소별 소개]
    카드 6개 (푸른수목원 / 항동도서관 / KB숲교육센터 / 항동철길 / 오류버들마을 / 구두인관)
    각 카드: 사진(placeholder) + 이름 + 매력포인트 + 방문 팁

[⑤ 맛집 & 먹거리]
    오류버들마을 대표 메뉴 카드 (감자옹심이 / 수제버거 / 통닭 / 브런치 등)

[⑥ 방문 정보]
    교통 안내 (버스 노선) + 주의사항 + 추천 시간대

[⑦ SNS 공유]
    해시태그 박스 + 공유 버튼

[푸터]
    답사 출처 | GLOWIDE
```

---

## 8. 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| 언어 | HTML + CSS + JavaScript | 배포 쉽고 별도 서버 불필요 |
| 지도 | 정적 SVG 일러스트 | API 키 없이 구현 가능, 나중에 카카오맵으로 교체 가능 |
| 사진 | 플레이스홀더 (색상 블록) | 실사진 준비 후 교체 |
| 배포 | GitHub Pages / Netlify | 무료, HTML 파일 업로드로 즉시 배포 |

---

## 9. 성공 기준

- 홈페이지 열었을 때 **30초 안에 코스 흐름**을 파악할 수 있다
- 스마트폰에서도 **불편 없이 볼 수 있다** (모바일 반응형)
- SNS 공유 버튼이 **실제로 동작**한다
