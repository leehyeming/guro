[실제 이미지 교체 매핑 가이드]

이 폴더(images/)에 가지고 계신 실제 구로구 수목원/철길/맛집 사진을 아래 파일명과 매치하여 넣어주시면, 코드 변경이 쉽거나 자동으로 교체되도록 구성할 수 있습니다.

1. 권장 이미지 규격 및 포맷
   - 포맷: .jpg, .png 또는 모바일 웹 최적화를 위해 용량이 가벼운 **.webp** 포맷을 강력히 권장합니다.
   - 용량 최적화: 가급적 이미지당 500KB 이하로 최적화해야 사이트 로딩 속도가 빨라집니다.

2. 매핑 파일명 목록:
   - hero.jpg        : 히어로 상단 배경 화면 (1920x1080 권장)
   - og-preview.jpg  : 카카오톡/인스타 공유용 미리보기 이미지 (1200x630 권장)
   - place-1.jpg     : 푸른수목원 소개 이미지
   - place-2.jpg     : 항동푸른도서관 소개 이미지
   - place-3.jpg     : KB숲교육센터 소개 이미지
   - place-4.jpg     : 항동철길 소개 이미지
   - place-5.jpg     : 오류버들마을 소개 이미지
